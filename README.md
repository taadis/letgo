# learn-tensorflow

## Python 环境

创建python虚拟环境

```
make .venv
```

## 安装 TensorFlow

```
pip install tensorflow
```

## or安装TensorFlow for Apple Silicon(M系列)

# 安装基础依赖
pip install tensorflow-macos

# 安装Metal插件（GPU加速）
pip install tensorflow-metal

# 可选：安装其他常用库
pip install numpy pandas matplotlib jupyter

> 上面前2个包都需要安装，不然macos识别不出来的GPU信息...
