堆栈

## 1. 堆栈的定义

栈的定义：栈是插入和删除只能在其一端进行操作的线性表，是一种操作受限的线性表。
栈顶：进行插入和删除一端。
栈低：封闭的另一端。
空栈：表中没有元素。

## 堆栈的基本操作

1. 栈初始化
2. 进栈 Push
3. 出栈 Pop
4. 读取栈顶元素 Peek
5. 判断栈是否为空 StackEmpty
6. 判断栈是否已满 StackFull
7. 置空栈

```
// 向栈顶压入一个元素
bool Push(item)

// 从栈顶弹出一个元素
bool Pop(item)

// 存取栈顶元素
bool Peek(item)

// 判断栈是否为空
int IsEmpty()

// 判断栈是否已满
int IsFull()

// 清空栈
void Clear()

```
